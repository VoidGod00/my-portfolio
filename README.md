A sleek, responsive, and interactive personal portfolio website designed to showcase my projects, technical skills, and experience as a Backend Developer specializing in Java, Spring Boot, and Cloud-Native Applications.

## ✨ Key Features
* **Modern UI/UX:** Dark, bold design with clean typography and striking orange accents.
* **Floating Navigation:** Context-aware floating navigation bar that dynamically highlights the active section as you scroll.
* **Smooth Animations:** Integrated scroll observer logic for dynamic, staggered entry effects on project and skill cards.
* **Fully Responsive:** Flexible Flexbox and CSS Grid layouts optimized for mobile, tablet, and desktop viewing.
* **Functional Contact Form:** Built-in message routing using EmailJS, allowing recruiters and peers to contact me directly from the site without requiring a dedicated backend server.
* **Performance Focused:** Lightweight static site utilizing semantic HTML5, custom CSS3, and vanilla JavaScript.

## 🛠️ Technologies Used
* **Markup & Styling:** HTML5, CSS3, CSS Variables (Custom Properties)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Integrations:** EmailJS (Form Handling), FontAwesome 6 (Icons), Google Fonts (Space Grotesk & Syne)
